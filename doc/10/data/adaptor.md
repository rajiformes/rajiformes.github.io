# Data Adaptors

Rajiformes by default works with **MongoDB** to persisting any type of data.

However Rajiformes allow to implementing database adaptors for any type database (relational, search, document, even file system :) )

>Developed MongoDB adapters are:

Adaptor | Description
--- | ---
**Schema** | For persisting system, entity, process defitions.
**Item** | For persisting master data related with process.
**Item Versions** | For persisting Item versions thoose are created with any modification on item
**Process Information** | All process instances, instance active states, and process journey logs
**System Logs** | All type logs about generated by running system
